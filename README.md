# Lewis's Medical-Surgical Nursing 12e

An [Astro.build](https://astro.build) website deployed to [GitHub Pages](https://sagreenxyz.github.io/lewis-med-surgical/) that serves as a digital study guide and reference companion for **Lewis's Medical-Surgical Nursing: Assessment and Management of Clinical Problems, 12th Edition**.

## Features

- 📖 Full chapter index organized by unit (Units 1–12, Chapters 1–64)
- 🩺 Study resources: nursing frameworks, pharmacology themes, lab value reference, clinical skills
- ⚡ Static site built with Astro — fast, accessible, deployable to GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Start dev server (http://localhost:4321)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The site is automatically deployed to GitHub Pages via the workflow in `.github/workflows/deploy.yml` whenever changes are pushed to the `main` branch.

Live URL: **https://sagreenxyz.github.io/lewis-med-surgical/**
